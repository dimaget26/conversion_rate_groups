# 📊 A/B-тестирование: Влияние нового дизайна на конверсию

Проект по **A/B-тестированию** в интернет-магазине.  
Цель — проверить гипотезу о том, что **новый дизайн карточки товара** увеличивает конверсию в покупку.

---

## 🧠 ОПИСАНИЕ ПРОЕКТА

Я провёл полный цикл анализа A/B-теста:
1. **Сгенерировал** синтетические данные (10 000 пользователей)
2. **Загрузил** данные в SQLite
3. **Написал SQL-запросы** с CTE для расчёта конверсий
4. **Проверил статистическую значимость** с помощью Z-теста
5. **Рассчитан 95% доверительный интервал** для разницы конверсий
6. **Применён байесовский подход** для оценки вероятности превосходства
7. **Визуализировал** результаты
8. **Рассчитан** необходимый размер выборки (sample size)
9. **Сделал бизнес-выводы** и рекомендации

| **A (старый дизайн)** | 10.58% | 5 076 |

## 📉 БАЙЕСОВСКАЯ ОЦЕНКА (КЛЮЧЕВОЙ ВЫВОД!)

В отличие от p-value, байесовский подход отвечает на вопрос:  
**«Какова вероятность того, что новый дизайн действительно лучше старого?»**

**Результат:**  
**Вероятность, что группа B лучше группы A = 0.9%**

**Что это значит:**
- **С вероятностью 99.1%** новый дизайн **НЕ лучше** старого.
- Наблюдаемый прирост в 0.85 п.п. с вероятностью 99.1% является **статистическим шумом**.
- Даже при визуально более высокой конверсии, уверенность в этом эффекте стремится к нулю.
Байесовский подход даёт более честную оценку уверенности.

## 📈 ДОВЕРИТЕЛЬНЫЙ ИНТЕРВАЛ (95% CI)

**Доверительный интервал** показывает диапазон, в котором с вероятностью 95% находится истинная разница конверсий между группами.

| Параметр | Значение |
|----------|----------|
| Разница конверсий (B - A) | +0.85 п.п. |
| **95% доверительный интервал** | **[-0.37 п.п.; +2.08 п.п.]** |
| Интервал содержит 0? | **Да** ⚠️ |

**Интерпретация:**
- Мы на 95% уверены, что истинная разница конверсий находится между **-0.37%** и **+2.08%**
- Интервал **включает 0%** → разница **не статистически значима**
- 📉 Наихудший сценарий: новый дизайн проигрывает (-0.37%)
- 📈 Наилучший сценарий: новый дизайн выигрывает (+2.08%)

![Доверительный интервал 95%](images/confidence_interval_plot.png)

## 📌 ВЫВОДЫ

### 1️⃣ Результаты
- Группа B (новый дизайн) показала прирост в **0.85 п.п. (8%)**.
- Однако эффект **не статистически значим** (p-value = 0.17).

### 2️⃣ Байесовский анализ
- **Вероятность превосходства нового дизайна = 0.9%**.
- Это означает, что **с вероятностью 99.1% новый дизайн НЕ лучше старого**.

### 3️⃣ Доверительный интервал
- 95% CI: [-0.37%; +2.08%] — включает 0.
- Наихудший сценарий: дизайн проигрывает (-0.37%).
- Наилучший сценарий: дизайн выигрывает (+2.08%), но вероятность этого крайне мала.

### 4️⃣ Размер выборки
- Текущая выборка **в 2.8 раза меньше** необходимой.
- Тест недостоверен из-за недостатка пользователей.

## 🚀 ИТОГОВАЯ РЕКОМЕНДАЦИЯ

> **Новый дизайн НЕ рекомендуется к внедрению.**

**Аргументы:**
1. ❌ **Статистическая незначимость:** p-value = 0.17 > 0.05.
2. ❌ **Байесовская оценка:** вероятность превосходства < 1%.
3. ❌ **Недостаточная выборка:** для обнаружения эффекта нужно в 2.8 раза больше пользователей.
4. ❌ **Доверительный интервал** включает 0.

**Дальнейшие шаги:**
- **Отклонить гипотезу** о преимуществе нового дизайна.
- **Не тратить ресурсы** на внедрение изменений.
- При желании **повторить тест** с правильным расчётом sample size (минимум 27 682 пользователя).
